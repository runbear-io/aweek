# Install

aweek ships as a Claude Code plugin.

## Requirements

- macOS 10.15 (Catalina) or newer. Linux and Windows aren't
  supported yet.
- Node.js 20 or 22.
- An active Claude Code session.

## Install

```bash
/plugin install aweek@runbear-io
```

The plugin's `SessionStart` hook installs the `aweek` CLI from npm
on first launch.
